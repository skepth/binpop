# BINPOP

## Usage

```
binpop is a binary exploration tool

Usage:
  binpop [command]

Available Commands:
  export-symbols export symbols from shared library
  help           Help about any command

Flags:
  -h, --help   help for binpop

Use "binpop [command] --help" for more information about a command.
```
